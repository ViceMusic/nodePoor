# 线程池模块指南

该模块旨在完成用户和对应的线程, 以及启动模型的效果

前端向node端发送ws请求, 即可构建链接, 链接完成构建以后, 会执行一个子线程模块, 该线程模块会运行python脚本. 其实核心来说就是每个前端可以顺利与独立的模型进行交互而不产生错误.

数据交互的方式目前比较简单正在完善

前端问一句, 脚本就会返回一句
前端宣布终止的时候, 后端的线程, 链接以及脚本全都会停止
前端启动的时候, 就开始生成一整套的链接了
